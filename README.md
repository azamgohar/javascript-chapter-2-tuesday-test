# JavaScript Practice Tasks

### Q1: (If / Else – Basic Check)
👉 Ek variable age banao aur usmein koi bhi number daalo
👉 Ab check karo:
* Agar age 18 ya usse zyada ho → "You can vote" print karo
* Warna → "You cannot vote" print karo

**(Hint: ek if ek else lagega.)**

---

### Q2: (If / Else If – Multiple Conditions)
👉 Ek variable marks banao
👉 Conditions likho:
* Agar marks 80 se zyada ho → "A Grade"
* Agar marks 50 se zyada ho → "Pass"
* Warna → "Fail"

**(Hint: multiple if aur ek else lagega kaam ho jaayga.)**

---

### Q3: (Truthy / Falsy Check)
👉 Ek variable value banao (ek ek karke ye teeno values daalo: 0, "hello", 22.)
👉 Ab sirf simple console log per apne words mein rint kardo ke in teeno mein se kon truthy hai aur kon falsy.

### Q4: (Type Coercion – + Operator)
👉 Ek variable a banao aur usmein "5" (string) daalo
👉 Ek variable b banao aur usmein 2 (number) daalo
👉 Ab dono ko add karo aur result console pe print karo
